🚦 Plot Status Stages

The stages a plot moves through, in order. Field Conductors report against these daily on the FC Portal's Plot Status. Set the ideal days a plot should stay in each stage before moving to the next — the day counter (and the coming over-time alert) uses it.

Deleting a stage keeps its name on past daily entries — only the reusable stage is removed.
